### [海盗湾] 磁力链接批量获取

 ![screenshot1](https://github.com/obaby/The-Pirate-Bay-MagnetLink-Extractor/blob/master/screenshot1.png)
---
这个工具的用途就是批量获取海盗湾的磁力链接，例如：https://thepiratebay.cr/search/tokyo%20hot

 ![screenshot2](https://github.com/obaby/The-Pirate-Bay-MagnetLink-Extractor/blob/master/screenshot2.png)
---
如果要获取链接使用迅雷下载可以使用这个工具，查看网页源代码，贴入上面的文本框，点击提取链接就会获取全部磁力连接了。
